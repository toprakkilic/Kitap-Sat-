| Katman   | URL                    |
|----------|------------------------|
| Frontend | http://localhost:3000 |
| Backend  | http://localhost:3001 |

### 💾 Veri Yönetimi

- In-memory veri yapıları  
- JSON tabanlı veri saklama  

---

## 🚀 **Kurulum ve Çalıştırma**

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyin:

### 🔧 1. Backend Kurulumu

```bash
cd backend
npm install
npm run start:dev
```

---

### 🎨 2. Frontend Kurulumu

```bash
cd frontend
npm install
npm start
```
